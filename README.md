To run on your local machine

```
git clone https://github.com/VERSAYANA/markdown-previewer.git
cd markdown-previewer
yarn install
yarn start
```

or

```
git clone https://github.com/VERSAYANA/markdown-previewer.git
cd markdown-previewer
npm install
npm start
```
